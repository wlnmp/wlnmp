# WLNMP - 一键安装包

wlnmp一键安装包基于上游开源软件二次开发，可以在Linux系统上通过wlnmp提供的镜像源，快速部署Nginx/Mysql/PHP等常用软件。

## wlnmp.sh脚本

- 自动识别当前系统以及版本
- 自动安装对应的WLNMP源

## wlnmp.sh脚本安装

- curl -fsSL "https://sh.wlnmp.com/wlnmp.sh" | bash
